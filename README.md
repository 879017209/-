# 西部创源项目

这个项目现在分成两层：

## 当前展示版

根目录这三个文件可以直接发布给别人看：

- `index.html`
- `styles.css`
- `app.js`

它们是静态网页，适合快速演示。

## 后续应用版

`future-app` 是后续升级用的 Next.js 项目底座。以后如果要加这些能力，就在这里继续做：

- 登录和成员权限
- 数据库保存任务
- 领取、提交、审核等后台接口
- 文件上传和存储
- 线上自动部署

推荐部署路线：Next.js + Vercel + Supabase。

## 一键预览后续应用版

双击根目录里的 `Start Future App.cmd`，启动后打开：

```text
http://127.0.0.1:3000
```
